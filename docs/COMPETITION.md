## 3. 신청서 요약 문장

신청서에 그대로 쓸 수 있는 표준 문장:

> 에비던스 베이스는 건강 데이터와 생명의학 근거를 바탕으로 개인 맞춤 건강 행동지원을 제공하는 AI 헬스케어 플랫폼 기업입니다. 본 대회에서는 산림 공공데이터, 기상·대기 공공데이터, 숲길·산림치유 공간 데이터를 융합하여 사용자의 건강 관심사와 체력 수준에 맞는 산림치유 공간을 추천하는 '회복의 숲' 서비스를 제안합니다. 한국통합치유협동조합의 산림치유 프로그램 운영 경험을 바탕으로 향후 현장 실증과 사용자 피드백 수집이 가능하며, 이를 통해 데이터 기반 산림복지 서비스로 확장하고자 합니다.

## 4. 평가항목 매핑

| 평가항목 | 회복의 숲 대응 |
|---|---|
| 공공데이터 활용도 | 산림청 5종(치유의숲·둘레길·자연휴양림·도시숲·유아숲), 기상청 단기예보, 에어코리아 실시간 미세먼지를 단일 추천에 융합 |
| AI 기술 활용 | LLM 기반 추천 이유/주의사항 생성, JSON 강제, 안전 필터, 4곳 치유의숲 피톤치드 실측 기반 전국 예측 모델 |
| AI 서비스 완성도 | 입력→결과 전체 자동화 (n8n 오케스트레이션), 5축 점수 시각화, 모바일 우선 UX |
| 독창성 | 산림 자원과 건강 관심사를 매칭하는 회복점수 모델, 4곳 치유의숲 데이터를 전국 산림에 외삽 |
| 발전 가능성 | 산림청·지자체·기업 산림복지 B2B, Evidence Base 플랫폼 통합, 산림치유 프로그램 예약/결제 확장 |
| ESG 혁신 | 산림복지 격차 해소, 회복기 사용자·중장년·장애인 접근성 강조, 도시-숲 연결 |

# Shadcn UI 컴포넌트 사용하기

MDX 포스트에서 Shadcn UI 컴포넌트를 사용할 수 있습니다. 이를 통해 더 풍부하고 일관된 디자인의 콘텐츠를 만들 수 있습니다.

## 1. Badge 컴포넌트

<Badge variant="default">기본 배지</Badge>
<Badge variant="secondary">보조 배지</Badge>
<Badge variant="destructive">경고 배지</Badge>
<Badge variant="outline">아웃라인 배지</Badge>

## 2. Button 컴포넌트

<Button>기본 버튼</Button>
<Button variant="outline">아웃라인 버튼</Button>
<Button variant="destructive">삭제 버튼</Button>
<Button size="sm">작은 버튼</Button>

## 3. Card 컴포넌트

<Card>
  <CardHeader>
    <CardTitle>카드 제목</CardTitle>
  </CardHeader>
  <CardContent>
    <p>카드 내용입니다. 여기에 다양한 콘텐츠를 넣을 수 있습니다.</p>
  </CardContent>
</Card>

## 4. Alert 컴포넌트

<Alert>
  <AlertTitle>정보 알림</AlertTitle>
  <AlertDescription>
    이것은 정보를 전달하는 알림 컴포넌트입니다.
  </AlertDescription>
</Alert>

<Alert variant="destructive">
  <AlertTitle>경고 알림</AlertTitle>
  <AlertDescription>이것은 중요한 경고를 전달하는 알림입니다.</AlertDescription>
</Alert>

## 5. Separator 컴포넌트

섹션을 구분할 때 사용합니다.

<Separator />

## 6. 조합 사용 예시

여러 컴포넌트를 조합해서 사용할 수도 있습니다:

<Card>
  <CardHeader>
    <CardTitle className="flex items-center gap-2">
      <Badge variant="secondary">NEW</Badge>
      기능 소개
    </CardTitle>
  </CardHeader>
  <CardContent className="space-y-4">
    <p>새로운 기능에 대한 설명입니다.</p>
    <Alert>
      <AlertDescription>이 기능은 아직 베타 버전입니다.</AlertDescription>
    </Alert>
    <div className="flex gap-2">
      <Button size="sm">자세히 보기</Button>
      <Button variant="outline" size="sm">
        문서 보기
      </Button>
    </div>
  </CardContent>
</Card>

## 주의사항

1. **컴포넌트 이름**: 정확한 컴포넌트 이름을 사용해야 합니다.
2. **Props**: 각 컴포넌트의 props를 올바르게 전달해야 합니다.
3. **스타일링**: className을 통해 추가 스타일링이 가능합니다.
4. **접근성**: 컴포넌트들이 접근성 표준을 준수합니다.

이렇게 Shadcn UI 컴포넌트를 활용하면 일관된 디자인과 풍부한 사용자 경험을 제공할 수 있습니다.
